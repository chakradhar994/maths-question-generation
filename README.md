# Maths Question Generation

This project generates mathematical questions using machine learning techniques.

## Features
- Dataset preprocessing
- Training ML model for question generation
- Combining multiple datasets
- Evaluation using test data

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Project Structure
preprocessing.py – data preprocessing  
train_model.py – model training  
combine_codes – dataset merging scripts  
combined_datasets – processed datasets
